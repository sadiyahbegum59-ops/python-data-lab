# Python Data Lab Practice

This repository contains beginner Python programs created during Data Intelligence and AI lab sessions.

## Topics Covered

- Lists
- Dictionaries
- Loops
- Conditional Statements
- Filtering Data
- Data Processing

## Programs Included

1. Print names of all students  
2. Calculate total marks  
3. Find highest marks student  
4. Filter students with marks greater than or equal to 80  
5. Add grades based on marks  

## Tools Used

- Python 3
- Jupyter Notebook
